# AASK Rehab & Physical Therapy - Employment Application System

Employment intake form system with AI-powered applicant analysis for AASK Rehab & Physical Therapy.

## Features
- Employment application form
- Admin panel with password protection
- AI analysis of applicants
- Export to CSV/Excel
- QR code generation

## Setup
1. Install dependencies: `npm install`
2. Configure `.env.local` with your Supabase and OpenAI keys
3. Run development server: `npm run dev`
4. Access admin panel: http://localhost:3000/admin (Password: TRDAASK)

## Tech Stack
- Next.js 14
- Supabase
- OpenAI
- TypeScript
- Tailwind CSS
